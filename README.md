---

## 🚀 Cómo Usar
1. **Agrega participantes** escribiendo nombres y pulsando <kbd>Añadir</kbd>
2. **Elimina participantes** haciendo clic en el botón `×`
3. **Realiza el sorteo** pulsando el botón principal
4. **¡Descubre** quién fue seleccionado!
